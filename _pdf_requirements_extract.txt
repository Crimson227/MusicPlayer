1.1 大作业题目
题目建议写：
基于 Android 的本地音乐播放器设计与实现
需要说明本项目是一个 Android 本地音乐播放器 App，主要面向本地音乐文件的导入、播放、资料库管理、搜
索、收藏、歌词和封面资源管理等场景。
1.2 项目背景与设计目标
说明为什么选择音乐播放器作为课程大作业。
可以写：
音乐播放器是移动端常见应用类型，能综合体现界面设计、文件读取、状态管理、媒体播放、用户交互
等移动开发能力。
本项目不是在线音乐平台，而是本地音乐播放器。
项目重点不是接入复杂网络服务，而是完成一个稳定、可运行、功能闭环完整、界面结构清晰的
Android 应用。
系统目标包括：
支持本地音乐导入
支持播放、暂停、停止、上一首、下一首
支持播放进度控制
支持播放列表管理
支持搜索歌曲
支持喜欢歌曲
支持最近播放记录
支持歌词、封面、背景等音乐资源管理
支持设置页面和基础播放偏好管理
支持状态持久化，重启后恢复数据
1.3 开发环境与技术选型
用表格写清楚：
类别 内容 说明
开发平台 Android 面向移动端设备
开发语言 Java 主要逻辑使用 Java 实现
构建工具 Gradle / Android Gradle Plugin用于构建和打包 APK
播放组件 MediaPlayer 实现本地音频播放
数据保存 SharedPreferences + JSON保存歌曲、播放列表、收藏、最近播放、主题等状态
文件导入 ACTION_OPEN_DOCUMENT用于选择音频、歌词、封面和背景文件
界面实现 XML 布局 + Java 动态渲染 页面主体通过代码动态生成
